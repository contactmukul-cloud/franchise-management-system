# 🚀 Franchise Management System (FastAPI)

A FastAPI backend implementing JWT authentication, role-based access control, franchise management, and real-time notifications using WebSockets.

---

## 📌 Features

- JWT Authentication (Super Admin & Franchise)
- Role-based access control
- Franchise CRUD APIs
- Pagination, search, filtering
- Profile API
- WebSocket real-time notifications

---

## 🛠️ Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- WebSocket

---

## ▶️ Run Project

pip install -r requirements.txt

uvicorn app.main:app --reload
